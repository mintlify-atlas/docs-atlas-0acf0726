## Authentication

All student endpoints require:
- **Session-based authentication** via `req.session.userId`
- **Role:** `coder`
- **Onboarding status:** Must be completed (`first_login = false`)

If onboarding is incomplete, endpoints return `403 Forbidden` with redirect to `/onboarding`.

## Get Dashboard

<CodeGroup>
```bash cURL
curl -X GET https://api.kairo.com/api/coder/dashboard \
  -H "Cookie: session=abc123..."
```

```javascript JavaScript
const response = await fetch('/api/coder/dashboard', {
  credentials: 'include'
});
const data = await response.json();
```
</CodeGroup>

**GET** `/api/coder/dashboard`

Returns comprehensive dashboard data including user profile, soft skills assessment, learning progress, active plans, notifications, and risk flags.

## Get Module Milestones

<CodeGroup>
```bash cURL
curl -X GET https://api.kairo.com/api/coder/milestones \
  -H "Cookie: session=abc123..."
```

```javascript JavaScript
const response = await fetch('/api/coder/milestones', {
  credentials: 'include'
});
const { milestones } = await response.json();
```
</CodeGroup>

**GET** `/api/coder/milestones`

Retrieves weekly milestones with scores across all modules for the authenticated student.

### Response

<ResponseField name="milestones" type="array">
  Array of milestone objects ordered by week number
  
  <ResponseField name="module_name" type="string">
    Module name
  </ResponseField>
  
  <ResponseField name="week_number" type="number">
    Week number within module
  </ResponseField>
  
  <ResponseField name="week_name" type="string">
    Week title/name
  </ResponseField>
  
  <ResponseField name="average_score" type="number">
    Average score for that week
  </ResponseField>
</ResponseField>

### Example Response

```json
{
  "milestones": [
    {
      "module_name": "Bases de Datos",
      "week_number": 1,
      "week_name": "Introduction to SQL",
      "average_score": 92
    },
    {
      "module_name": "Bases de Datos",
      "week_number": 2,
      "week_name": "Database Design",
      "average_score": 88
    }
  ]
}
```

## Mark Feedback as Read

<CodeGroup>
```bash cURL
curl -X PATCH https://api.kairo.com/api/coder/feedback/101/read \
  -H "Cookie: session=abc123..."
```

```javascript JavaScript
const response = await fetch('/api/coder/feedback/101/read', {
  method: 'PATCH',
  credentials: 'include'
});
const data = await response.json();
```
</CodeGroup>

**PATCH** `/api/coder/feedback/:id/read`

Marks a Team Leader feedback notification as read and sends a real-time notification to the TL.

### Path Parameters

<ParamField path="id" type="integer" required>
  Feedback ID to mark as read
</ParamField>

### Response

<ResponseField name="success" type="boolean">
  Whether the operation succeeded
</ResponseField>

<ResponseField name="message" type="string">
  Success message
</ResponseField>

### Example Response

```json
{
  "success": true,
  "message": "Feedback marked as read"
}
```

### Error Responses

<ResponseField name="error" type="string">
  Error message when feedback not found or operation fails
</ResponseField>

```json 404
{
  "error": "Feedback not found"
}
```

```json 500
{
  "error": "Failed to mark feedback as read"
}
```
