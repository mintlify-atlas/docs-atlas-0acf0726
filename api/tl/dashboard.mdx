### Endpoint

```
GET /api/tl/dashboard
```

### Description

Retrieves comprehensive dashboard data for the Team Leader's clan, including:
- All students (coders) in the clan with their progress and soft skills assessments
- Aggregated clan-level metrics (average score, onboarding status, risk flags)
- Clan-wide soft skills averages calculated from completed assessments

### Authentication

Requires active session with `tl` (Team Leader) role.

### Response Fields

<ResponseField name="tl" type="object">
  Team Leader information
  
  <ResponseField name="fullName" type="string">
    Full name of the Team Leader
  </ResponseField>
  
  <ResponseField name="clanId" type="string">
    Clan identifier assigned to this Team Leader
  </ResponseField>
</ResponseField>

<ResponseField name="coders" type="array">
  Array of all students in the clan
  
  <ResponseField name="id" type="integer">
    Student user ID
  </ResponseField>
  
  <ResponseField name="full_name" type="string">
    Student's full name
  </ResponseField>
  
  <ResponseField name="email" type="string">
    Student's email address
  </ResponseField>
  
  <ResponseField name="first_login" type="boolean | null">
    Onboarding status. `null` if completed, `true` if pending
  </ResponseField>
  
  <ResponseField name="clan_id" type="string">
    Clan identifier
  </ResponseField>
  
  <ResponseField name="average_score" type="number | null">
    Average Moodle quiz score across all completed weeks
  </ResponseField>
  
  <ResponseField name="current_week" type="integer | null">
    Current week of course progression
  </ResponseField>
  
  <ResponseField name="autonomy" type="number | null">
    Autonomy soft skill rating (1-5)
  </ResponseField>
  
  <ResponseField name="time_management" type="number | null">
    Time management soft skill rating (1-5)
  </ResponseField>
  
  <ResponseField name="problem_solving" type="number | null">
    Problem solving soft skill rating (1-5)
  </ResponseField>
  
  <ResponseField name="communication" type="number | null">
    Communication soft skill rating (1-5)
  </ResponseField>
  
  <ResponseField name="teamwork" type="number | null">
    Teamwork soft skill rating (1-5)
  </ResponseField>
  
  <ResponseField name="learning_style" type="string | null">
    Identified learning style: `visual`, `auditory`, `kinesthetic`, or `reading_writing`
  </ResponseField>
  
  <ResponseField name="risk_level" type="string | null">
    Active risk flag level: `critical`, `high`, `medium`, or `null` if no active risk
  </ResponseField>
</ResponseField>

<ResponseField name="overview" type="object">
  Aggregated clan-level metrics
  
  <ResponseField name="totalCoders" type="integer">
    Total number of students in the clan
  </ResponseField>
  
  <ResponseField name="completedOnboarding" type="integer">
    Number of students who completed onboarding (`first_login = null`)

  <ResponseField name="pendingOnboarding" type="integer">

  <ResponseField name="clanAvgScore" type="string">

</ResponseField>

<ResponseField name="softSkillsAverage" type="object">
  Clan-wide soft skills averages (only from students who completed the assessment)
  
  <ResponseField name="autonomy" type="number">

  <ResponseField name="time_management" type="number">


### Implementation Notes

- Uses LEFT JOIN to include students who haven't completed onboarding or soft skills assessment
- Soft skills averages exclude students who haven't completed the assessment to avoid skewing data
- Risk levels are filtered to show only unresolved flags
- Students are ordered alphabetically by full name
