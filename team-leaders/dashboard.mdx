## Overview

The Team Leader Dashboard is your command center for monitoring all students (coders) in your clan. It provides real-time insights into performance, soft skills, and learning progress.

<Note>
  The dashboard automatically refreshes when new feedback is read or system updates occur.
</Note>

## Accessing the Dashboard

Navigate to your dashboard at `/tl/dashboard` after logging in with your Team Leader credentials.

**API Endpoint:** `GET /api/tl/dashboard`

### 2. Soft Skills Overview

View aggregated soft skills data for your entire clan:

- **Autonomy** - Self-directed learning capability
- **Time Management** - Ability to organize and prioritize tasks
- **Problem Solving** - Critical thinking and debugging skills
- **Communication** - Clarity in expressing technical concepts
- **Teamwork** - Collaboration and peer support

Each skill is rated on a scale of 0-5 and displayed with visual progress bars.

<Accordion title="How are soft skills calculated?">
  Soft skills averages are calculated only from students who have completed the diagnostic assessment. This prevents skewing the data toward zero for students who haven't been assessed yet.

  **Backend Logic:**
  ```javascript
  const codersWithSkills = coders.filter((c) => c.learning_style !== null);
  softSkillsAverage[skill] = total / codersWithSkills.length;
  ```
</Accordion>

### 3. Learning Styles Distribution

A visual pie chart shows the distribution of learning styles in your clan:

- **Visual** - Learn best through diagrams and visual aids
- **Auditory** - Prefer lectures and verbal explanations
- **Kinesthetic** - Hands-on coding and experimentation
- **Read/Write** - Text-based learning through documentation
- **Not Diagnosed** - Haven't completed the assessment yet

## Data Source

The dashboard aggregates data from multiple database tables:

**Backend Query (tlControllers.js:248)**
```javascript
SELECT 
  u.id, u.full_name, u.email, u.first_login, u.clan AS clan_id,
  mp.average_score, mp.current_week,
  ss.autonomy, ss.time_management, ss.problem_solving, 
  ss.communication, ss.teamwork, ss.learning_style,
  rf.risk_level
FROM users u
LEFT JOIN moodle_progress mp ON u.id = mp.coder_id
LEFT JOIN soft_skills_assessment ss ON u.id = ss.coder_id
LEFT JOIN risk_flags rf ON u.id = rf.coder_id AND rf.resolved = false
WHERE u.role = 'coder' AND u.clan = $1
```
