# Welcome to Kairo Learning Platform

Kairo is an intelligent ecosystem designed to bridge the gap between intensive bootcamps and individual learning paces. By leveraging a hybrid architecture of Node.js and Python, the platform transforms academic data and soft skills assessments into dynamic, personalized 4-week growth paths.

## The Vision

Riwi's intensive 11-month program moves at a lightning pace. Kairo ensures no student is left behind by analyzing technical performance and soft skills to build a curriculum that adapts to the human, not just the code.

## Key Solution Pillars

<AccordionGroup>
  <Accordion title="Adaptive Learning" icon="brain">
    Generates custom 4-week study plans based on Moodle data and self-assessments. The AI analyzes student performance patterns and creates personalized learning paths that adapt to individual needs.
  </Accordion>
  
  <Accordion title="Holistic Growth" icon="user-graduate">
    Integrates soft skills (emotional intelligence, communication) directly into technical reinforcement tasks. Students develop both technical and interpersonal competencies simultaneously.
  </Accordion>
  
  <Accordion title="Proactive Mentorship" icon="chart-line">
    Provides Team Leaders with an analytics dashboard to identify at-risk students before academic failure occurs. Real-time risk detection enables timely interventions.
  </Accordion>
  
  <Accordion title="Hybrid Intelligence" icon="microchip">
    Uses Node.js for robust orchestration and Python/FastAPI for advanced local AI processing with Qwen2.5-7B-Instruct model.
  </Accordion>
</AccordionGroup>

## Tech Stack

### Frontend
- **HTML5 / CSS3 / JavaScript (Vanilla)** — Core structure and interactivity
- **Bootstrap 5** — Responsive layout and rapid UI components
- **EJS Templates** — Server-side rendering for dynamic content

### Backend (Hybrid Architecture)
- **Node.js + Express** — Main server managing users, authentication, and business logic
- **Python + FastAPI** — AI microservice for LLM inference and analytics

### Artificial Intelligence
- **Model** — Qwen2.5-7B-Instruct (via Groq)
- **Orchestrator** — LangChain for prompt management and JSON validation

### Database
- **PostgreSQL (Supabase)** — Storage for user profiles, learning plans, and analytics
- **Custom Logic** — Specialized enumerated types for roles, risk levels, and AI agents
